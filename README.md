# Shell_Installer

## INSTALL
To use it:
```sh
git clone https://github.com/belotte55/Shell_Installer.git
cd Shell_Installeur
bash install.sh
```
The installer can create a .zshrc file, which works with Z Shell (zsh).
To install it:
```
chsh -s $(which zsh)
```
